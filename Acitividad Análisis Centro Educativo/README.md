## Actividad. Análisis del centro educativo

En esta actividad, el alumnado realizará una primera aproximación al concepto de edificio inteligente y sostenible mediante el análisis de su propio centro educativo.

Organizados en equipos, los estudiantes utilizarán el **Mapa Tecnológico del Centro** para localizar diferentes zonas estratégicas del instituto. En cada una de ellas encontrarán una tarjeta con pistas que les ayudará a identificar posibles necesidades de mejora relacionadas con la eficiencia energética, la sostenibilidad y la automatización.

A partir de las pistas obtenidas, cada grupo debatirá qué problemas o aspectos podrían optimizarse e investigará, utilizando fuentes fiables, diferentes tecnologías y soluciones que permitan dar respuesta a las necesidades detectadas.

Finalmente, toda la información recopilada se reflejará en la **Ficha de análisis del centro educativo**, donde deberán definir los conceptos fundamentales, registrar las deficiencias encontradas, relacionarlas con tecnologías existentes y plantear las primeras propuestas de mejora. Este análisis constituirá la base para el desarrollo del proyecto que elaborarán durante las siguientes sesiones de la situación de aprendizaje.
